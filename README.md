🔒 LeakLens: Global Breach Analytics
📌 Project Overview
LeakLens is a high-performance web application that demonstrates advanced JavaScript skills and API integration. It uses the Have I Been Pwned (HIBP) Breaches API to provide a searchable and interactive view of global data breaches, helping users stay aware of cybersecurity risks.
🎯 Objective
Build a real-world project using Vanilla JavaScript
Practice working with public APIs
Implement Array Higher-Order Functions (HOFs)
Create a clean and responsive UI
🛠️ Tech Stack
HTML5 & CSS3
Vanilla JavaScript
HIBP API (Breaches Endpoint)
🚀 Features
🔍 Search

Search breaches by company/service name (e.g., Adobe)
🎯 Filtering
Filter breaches based on exposed data:
Passwords
Emails
Credit Cards
📊 Sorting
Sort breaches by:

Number of affected accounts (PwnCount)
Date of breach
⭐ Favorites
Save breaches using Local Storage
Access saved breaches anytime
🧩 Interactive UI
"View More" for detailed breach info
Responsive design
🌗 Theme Toggle
Switch between Dark Mode and Light Mode
⚙️ Implementation Details
Uses fetch() for API calls
Uses HOFs:
map()
filter()
sort()
Dynamic DOM updates for real-time interaction
